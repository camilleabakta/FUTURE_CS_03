# Secure File Sharing System 

Projet académique en cybersécurité visant à concevoir un système de partage de fichiers sécurisé
en appliquant les principes fondamentaux de la cryptographie.

## Objectif
Garantir la confidentialité et la sécurité des fichiers échangés entre utilisateurs
grâce à des mécanismes de chiffrement robustes.

## Fonctionnalités
- Chiffrement des données avant partage
- Gestion sécurisée des clés
- Protection contre l’accès non autorisé
- Architecture orientée sécurité

## Technologies utilisées
- Python
- Cryptographie (concepts RSA / chiffrement asymétrique)
- Flask (si applicable)
- Sécurité applicative

## Lancement du projet
```bash
pip install -r requirements.txt
python app.py
```
